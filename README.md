## Featured Projects

### nik-parser.rb

Indonesian identity toolkit for Ruby. Parse NIK, validate NPWP, phone numbers, and format Rupiah. Built with love for the Indonesian developer community.

### isbn-toolkit

All-in-one ISBN toolkit. Validate ISBN-10/13, convert between formats, and lookup book metadata from Open Library and Google Books.

| Stack | Repo |
|---|---|
| Ruby | [github.com/gizipp/isbn-toolkit-ruby](https://github.com/gizipp/isbn-toolkit-ruby) |
| JavaScript/TypeScript | [github.com/gizipp/isbn_toolkit](https://github.com/gizipp/isbn_toolkit) |
| PHP (Laravel) | [github.com/gizipp/isbn-toolkit-php](https://github.com/gizipp/isbn-toolkit-php) |
